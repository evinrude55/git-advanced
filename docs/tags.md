## Creating tags

Tags are typically used to mark release points (v1.0, v2.0 and so on).

**Listing tags**

```shell
git tag
```

**Creating tags**

```shell
git tag -a v1 -m 'My first version'
```

**Creating lightweight tags**

```shell
git tag v1
```

**Display tags**

```shell
git show v1
```

**Delete tags**

```shell
git tag --delete <tag_name>
```
